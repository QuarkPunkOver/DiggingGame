# Digging Game - Core of the Earth

## 📋 О игре

**Digging Game** - это увлекательная игра о бурении к центру Земли. Исследуйте глубины, добывайте ресурсы, улучшайте свой бур и открывайте новые технологии. Сможете ли вы достичь ядра планеты?

### ✨ Особенности

- 🎮 **Простое и интуитивное управление**
- 🌍 **Процедурно генерируемый мир** - каждый заход уникален
- ⛏️ **Разнообразные ресурсы** - от обычного гравия до редких изотопов
- 🔧 **Система улучшений** - прокачивайте бур, корпус, топливную систему
- 🏭 **Специальные здания** - заправка, магазин, технический сервис
- 🌡️ **Температурная механика** - чем глубже, тем жарче
- 🎵 **Атмосферный саундтрек** - меняется с глубиной
- 🌍 **Локализация** - русский и английский языки

## 🎮 Геймплей

### Основная цель
Пробуриться к центру Земли, добывая ценные ресурсы и улучшая оборудование.

### Ресурсы

- **Обычные**: гравий, камень, гранит, уголь
- **Металлы**: медь, олово, железо, золото, платина
- **Редкие**: вольфрам, уран, алмазы
- **Особые**: изотоп урана, мягкая материя, плотная материя
- **Легендарные**: фрагменты ядра

# Digging Game - Core of the Earth

## 📋 About the Game

**Digging Game** is an exciting game about drilling to the center of the Earth. Explore the depths, mine resources, upgrade your drill, and discover new technologies. Can you reach the planet's core?

### ✨ Features

- 🎮 **Simple and intuitive controls**
- 🌍 **Procedurally generated world** - every playthrough is unique
- ⛏️ **Diverse resources** - from ordinary gravel to rare isotopes
- 🔧 **Upgrade system** - improve your drill, hull, fuel system
- 🏭 **Special buildings** - fuel station, shop, tech service
- 🌡️ **Temperature mechanics** - the deeper, the hotter
- 🎵 **Atmospheric soundtrack** - changes with depth
- 🌍 **Localization** - Russian and English languages

## 🎮 Gameplay

### Main Objective
Drill to the center of the Earth, mining valuable resources and upgrading your equipment.

### Resources
The game features various resources:
- **Common**: gravel, stone, granite, coal
- **Metals**: copper, tin, iron, gold, platinum
- **Rare**: tungsten, uranium, diamonds
- **Special**: uranium isotope, soft matter, dense matter
- **Legendary**: core fragments

### Upgrades
| Upgrade | Effect | Cost |
|---------|--------|------|
| Diamond Coating | +66% drilling speed | 10 diamonds, $1000 |
| Electric Upgrade | +10% speed, -10% fuel | 20 gold, $500 |
| Uranium Engine | +50% power, -20% fuel | 50 uranium, 30 tungsten, $5000 |
| Titanium Drill | +30% power, -15% fuel | 40 tungsten, 20 platinum, $8000 |
| Plasma Cutter | +100% power, -30% fuel, +500°C | 30 uranium isotope, 25 diamonds, $15000 |
| Neutronium Core | +200% power, -50% fuel, +1000°C, +50% capacity | 5 core fragments, 50 dense matter, $50000 |

## 🎮 Controls

| Key | Action |
